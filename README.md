# Interactive Wall Calendar Component

A highly functional, responsive wall calendar component built with React (Vite) and Vanilla CSS, designed to resemble a physical wall calendar as per the Front-End Engineering Challenge.

## Features
- **Physical Wall Calendar Aesthetic**: Features a prominent hero image connected with a dynamic date overlay and spiral-bound ring styles.
- **Day Range Selector**: Select start and end dates with clear visual states and connecting background highlights.
- **Integrated Notes**: A functional notes area persistent across sessions via `localStorage`, styled creatively like physical lined paper.
- **Responsive Design**: Elegant Desktop grid that gracefully stacks on mobile devices.

## Running Locally

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server (runs on `http://localhost:5173`):
   ```bash
   npm run dev
   ```
3. Build for production (optional):
   ```bash
   npm run build
   ```
